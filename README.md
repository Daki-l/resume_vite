## ✨ 功能特性

- 📝 **3 套简历模板**，可视化切换
- 🎨 **自定义主题色** / 标签色
- 🖱️ **拖拽排序** 模块内容
- 📤 **导入/导出** JSON 配置
- 📄 **下载 PDF**（浏览器打印）
- ⚡ **Vite 构建**，秒级启动

## 🛠️ 技术栈

| 技术 | 说明 |
| --- | --- |
| React 18 | UI 框架 |
| Ant Design 5 | 组件库 |
| Vite 5 | 构建工具 |
| TypeScript | 类型安全 |
| Less | 样式预处理 |

## 🚀 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 生产构建
npm run build
```
